# smartnest
A personal finance web app for tracking transactions, budgets, and savings goals.
# Currently under development more features coming soon.
## Features
 Real-time expense tracking  
 Income & balance overview    
 Budget planner with progress bars  
 Savings goals tracker  
 Live activity feed  
 Interactive charts  
 Responsive design  
 Clean UI  
## Technologies Used
- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Chart.js
## Pages
- Dashboard
- Transactions
- Budgets
- Goals
## How It Works
All data is stored locally in the browser using LocalStorage, so no backend is required.
## Purpose
Portfolio project demonstrating front-end skills, UI design, and dynamic functionality.
